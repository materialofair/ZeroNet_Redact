## 3. App 描述（Description，≤4000 字符）

**中文**

```
ZeroNet Redact（零网隐私）是一款完全离线的隐私打码工具。发朋友圈、提交材料、分享聊天记录之前，把不该露出的信息安全地遮住。

【AI 自动识别】
内置离线文字识别引擎，自动找出图片和 PDF 中的手机号、邮箱、身份证号、银行卡号等敏感信息，逐条确认或一键全部打码，多页 PDF 也能按页处理。

【手动精细涂抹】
手指涂抹即可打码，支持捏合放大精确处理小字；提供马赛克、模糊、纯黑、彩色遮盖多种效果和三档画笔粗细；支持撤销/重做，导出前可随时调整已打码区域的位置和大小。

【真正的隐私保护】
• 完全离线：无网络请求，不需要注册账号，你的文件永远不会离开这台设备
• 本地加密：导入的原始文件全部加密存储（AES-GCM），支持密码 + Face ID / Touch ID 应用锁
• 防偷窥：切换到后台时自动遮挡界面，多任务切换器中不会泄露内容

【文件管理】
自定义分组管理原始文件与打码结果，打码后的文件可直接分享或导出，原图始终保留，随时重新处理。

【免费与升级】
免费版每天可导出 3 张图片和 3 个 PDF。一次性买断 Pro 即可解锁无限导出，无订阅、无广告。

适合场景：晒单/晒票据前遮住单号和金额、提交证件照片时隐藏无关信息、分享聊天记录前抹掉联系方式、发送合同 PDF 前遮盖签名与账号。

我们不收集任何数据——因为根本没有服务器。
```

**English**

```
ZeroNet Redact is a fully offline privacy redaction tool. Before you post, submit, or share a photo or PDF, cover up what shouldn't be seen — safely.

AI-POWERED DETECTION
The built-in offline text recognition engine automatically finds phone numbers, email addresses, ID numbers and bank card numbers in your photos and PDFs. Review each finding one by one, or redact them all in a single tap. Multi-page PDFs are handled page by page.

Just draw with your finger. Pinch to zoom for small text, choose mosaic, blur, solid black or colored block effects, pick from three brush sizes, and undo/redo freely. Move or resize applied redactions any time before export.

FREE & PRO

Perfect for: hiding order numbers on receipts, masking details on ID photos, removing contacts from chat screenshots, covering signatures and account numbers in contract PDFs.
```

## 5. 审核备注（App Review Notes）

**中文**

```
【应用说明】本应用是完全离线的图片/PDF 隐私打码工具，无服务器、无账号体系、不收集任何用户数据。

【测试路径】首次启动会提示设置应用锁密码，可点"稍后设置"跳过。核心流程：导入 Tab → 选择图片或 PDF → 进入编辑器 → 点击"AI 检测"自动识别敏感信息（或手指直接涂抹）→ 导出 → 在"脱敏文件"Tab 查看结果。

【解除免费限额（供审核测试）】免费版每天限导出 3 张图片和 3 个 PDF。如需不受限地完整测试，请在"设置"Tab 中连续点按页面顶部的应用图标 7 次，在弹出的"开发者选项"中输入访问码：REVIEW2026，即可解锁无限导出（该入口 2026 年 8 月 1 日后自动失效，仅用于审核）。

【内购】单个非消耗型项目 com.zeronet.redact.premium（解锁无限导出），可用沙盒账号正常购买与恢复购买。
```

**English**

```
ABOUT THIS APP: A fully offline photo/PDF privacy redaction tool. No server, no account system, no user data collected.

HOW TO TEST: On first launch you may skip the app-lock setup via "Set Up Later". Core flow: Import tab → pick a photo or PDF → editor opens → tap "AI Detect" to find sensitive info automatically (or draw with a finger) → Export → results appear in the "Redacted Files" tab.

LIFTING THE FREE-TIER LIMIT (for review): The free tier allows 3 image and 3 PDF exports per day. To test without limits, go to the Settings tab, tap the app icon at the top of the page 7 times, then enter access code REVIEW2026 in the "Developer Options" prompt. This unlocks unlimited exports (the entrance auto-expires on Aug 1, 2026 and exists only for review purposes).

IN-APP PURCHASE: One non-consumable item, com.zeronet.redact.premium (unlocks unlimited exports). Purchase and Restore both work with a sandbox account.
```
