## 🔍 常见问题排查

### 问题1: 还是会闪一下黑屏

**可能原因**：
- NavigationView 的背景色设置问题
- 系统暗黑模式影响

**解决方案**：
检查 `SimpleBrushEditor.swift:71` 是否有：
```swift
Color.white  // 白色底层
```

如果还有问题，添加：
```swift
.preferredColorScheme(.light)  // 强制浅色模式
```

### 问题2: 加载动画不显示

**可能原因**：
- `viewModel.isLoading` 状态更新有问题
- 主线程被阻塞

**排查方法**：
在 `EditorViewModel.swift:28` 添加日志：
```swift
await MainActor.run {
    isLoading = true
    print("🔄 isLoading = true")
}
```

在 `EditorViewModel.swift:32` 添加日志：
```swift
Task { @MainActor in
    isLoading = false
    print("🔄 isLoading = false")
}
```

### 问题3: 图片显示后有闪烁

**可能原因**：
- 多次 UI 更新
- 背景色切换

**解决方案**：
确保 `imageCanvas` 区域有稳定的背景：
```swift
.background(Color.white)
```

## 📊 性能基准

### 正常性能指标

| 指标 | 目标值 | 测试方法 |
|------|--------|---------|
| 小图片加载时间 | < 1秒 | 使用 500KB 图片测试 |
| 中等图片加载时间 | 1-3秒 | 使用 3MB 图片测试 |
| 大图片加载时间 | 3-5秒 | 使用 8MB 图片测试 |
| 缓存命中加载时间 | < 0.1秒 | 第二次打开同一图片 |
| UI 响应延迟 | < 100ms | 点击到显示加载动画 |

### 性能问题诊断

**如果加载时间超过10秒**：
1. 检查图片大小（是否超过10MB）
2. 检查解密性能（CPU占用）
3. 检查磁盘IO性能

**如果UI响应延迟超过500ms**：
1. 检查主线程是否被阻塞
2. 检查是否有同步IO操作
3. 使用 Instruments 分析性能瓶颈
